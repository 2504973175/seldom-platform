# django-ninja
基于django 的后端项目。

## 主要技术栈
* django
* django-ninja
* seldom

## 安装

* 安装依赖

```shell
> pip install -r requirements.txt
```

* 执行数据库同步

```
> python manage.py runserver
```

## 启动

```shell
> python manage.py runserver
```
